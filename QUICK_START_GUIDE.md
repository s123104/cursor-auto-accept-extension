# 🚀 Cursor Auto Accept Extension - 快速開始指南

> **📅 更新時間**：2025-06-11T17:04:06+08:00  
> **🔢 當前版本**：v1.0.1  
> **⏱️ 預估時間**：5 分鐘快速上手

## 🎯 立即體驗新控制面板

### 📦 步驟 1：快速安裝

#### 方法 A：VSIX 檔案安裝（推薦）⚡

1. **下載最新版本**

   - 獲取 `cursor-auto-accept-extension-1.0.1.vsix`（859,997 bytes）
   - 包含完整功能，即裝即用

2. **一鍵安裝**

   ```bash
   # 命令行安裝
   code --install-extension cursor-auto-accept-extension-1.0.1.vsix
   ```

   或在 VS Code 中：

   - 按 `Ctrl+Shift+P` → 輸入 "Extensions: Install from VSIX..."
   - 選擇下載的 `.vsix` 檔案
   - 重新載入 VS Code 視窗

#### 方法 B：開發模式（進階用戶）🔧

```bash
git clone https://github.com/s123104/cursor-auto-accept-extension.git
cd cursor-auto-accept-extension
npm install && npm run compile
# 在 VS Code 中按 F5 進入開發模式
```

### ⚙️ 推薦初始設定

```json
{
  "cursorAutoAccept.enabled": true,
  "cursorAutoAccept.interval": 2000,
  "cursorAutoAccept.enableAcceptAll": true,
  "cursorAutoAccept.enableAccept": true,
  "cursorAutoAccept.enableApply": true,
  "cursorAutoAccept.enableRun": false, // 🚨 建議手動控制執行操作
  "cursorAutoAccept.enableExecute": false, // 🚨 安全考量
  "cursorAutoAccept.debugMode": false
}
```

## 🔧 常見問題快速解決

### ❓ 控制面板沒有出現？

**🔍 檢查清單：**

- ✅ 確保在支援的檔案類型中（.js, .ts, .jsx, .tsx）
- ✅ 嘗試使用命令面板：`Ctrl+Shift+P` → "Show Control Panel"
- ✅ 檢查擴展是否已啟用：Extensions → 搜尋 "Cursor Auto Accept"

**💡 解決方案：**

```javascript
// 在 VS Code 開發者工具控制台中執行
globalThis.simpleAccept?.showControlPanel();
```

### ❓ 自動點擊沒有作用？

**🔍 診斷步驟：**

1. ✅ 檢查 Cursor 是否在前景執行
2. ✅ 確認控制面板狀態顯示「執行中」
3. ✅ 查看活動日誌是否有錯誤訊息
4. ✅ 確認相關按鈕類型已勾選啟用

**💡 快速修復：**

```
停止 → 重新開始 → 檢查設定 → 觀察日誌
```

### ❓ 如何自定義按鈕行為？

**🎛️ 配置步驟：**

1. 開啟控制面板主標籤
2. 點擊「設定」展開配置區域
3. 根據需求勾選/取消勾選按鈕類型
4. 變更會即時生效，無需重啟

### ❓ 資料會遺失嗎？

**💾 資料安全：**

- ✅ **自動保存**：所有資料儲存在 VS Code 全域狀態
- ✅ **手動備份**：定期從「分析」標籤匯出 JSON 備份
- ✅ **版本升級**：更新擴展時資料會自動保留
